# NZ Change Map Local Database

The local SQLite database is generated by:

```powershell
python tools\nz_change_map_pipeline.py
```

It writes:

- `data/nz-change-map/nz-change-map.sqlite` - local source-of-truth database
- `docs/nz-change-map/data.js` - static export consumed by GitHub Pages

Current pipelines are source-aware seed adapters. They store source URLs, pipeline names, and notes in `pipeline_runs` so live parsers can replace each seed adapter without changing the frontend data contract.

Useful tables:

- `regions`
- `datasets`
- `sources`
- `dataset_sources`
- `years`
- `observations`
- `pipeline_runs`

Open-source Twitch diagnostics add-on

Twitch Diagnostics Console

Advanced playback logging for Twitch streams: video CDN host, response timing, buffer health, live edge gap, throughput, dropped frames, and exportable JSON reports.

Firefox Add-on View Source on GitHub

What It Logs

Playback Health

Buffer ahead, live edge gap, playback state, dropped frames, and recent throughput.

Video CDN Diagnostics

Connected video CDN hosts, request counts, status codes, byte counts, and response times.

Exportable Reports

Copy or download a local JSON report for troubleshooting, support, or ISP routing evidence.

Diagnostics Only

Twitch Diagnostics Console observes local playback and network timing signals. It does not change Twitch playback, force CDNs, proxy traffic, bypass Twitch behavior, or transmit diagnostics to any server. The project is released under the MIT license.

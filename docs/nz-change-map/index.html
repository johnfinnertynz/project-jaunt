Build note

A public-data time machine for Aotearoa

This is a working prototype. The map, controls, and dataset model are real; the displayed regional values are seed data used to prove the interface before automated public-data pipelines are connected. The app is designed to ingest official open sources such as Stats NZ, MHUD, RBNZ, LCDB, RSM, and broadband rollout datasets.
